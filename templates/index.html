Exemples

Envoyez un message à Wahou et recevez instantanément une réponse précise sur les procédures et services du GUDE. L'historique de conversation est conservé 30 minutes par session pour un échange naturel et continu.

Corps de la requête
JSON — Request Body 
{
  "session_id": "user-abc-123",
  "message": "Quels sont les documents requis pour créer une PME ?"
}
200 OK 
{
  "session_id": "user-abc-123",
  "message": "Quels documents…",
  "response": "Pour créer une PME au GUDE…",
  "is_new_session": false
}
4xx / 5xx Erreur 
{
  "detail": "Le message ne peut
    pas être vide."
}
ParamètreTypeDescription
session_idREQUIS string Identifiant unique de session généré côté client (ex : UUID). Permet à Wahou de mémoriser le contexte de la conversation.
messageREQUIS string La question ou le message de l'utilisateur, en langage naturel. Doit concerner les procédures et services du GUDE.

Exemple d'appel à Wahou avec cURL :

cURL 
curl -X POST http://localhost:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"session_id":"user-abc-123","message":"C'\''est quoi le GUDE ?"}'
